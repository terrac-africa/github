Course Modules

From Git fundamentals to Enterprise Automation.

Module 1: Git Foundations

Understanding Distributed Version Control. Moving from legacy systems to Git. Configuring the local environment (User, Email, SSH).

Module 2: The GitHub Flow

Branching strategies, Pull Requests, Code Reviews, and merging. Handling conflicts in a team environment.

Module 3: GitHub Actions (CI/CD)

Automating workflows. Writing YAML files for testing, building, and deploying code automatically.

Module 4: Security & Governance

Secret Scanning, Dependabot, and managing permissions in an Enterprise organization.
